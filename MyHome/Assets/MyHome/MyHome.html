Hardware und Betriebssystem

Die App MyHome kann unter Windows 10 ausgeführt werden. Entweder auf einem PC, einem Notbook oder einem Tablet. MyHome ist jedoch dafür gedacht auf einem Rapberry Pi unter Windows 10 IoT installiert zu werden.

Um MyHome nutzen zu können sind einige Module individuell zu konfigurieren.

Modul Bilder: Anzeige der Bilder aus der Dropbox

  1. Gehe zu der Seite https://www.dropbox.com/developers und loge Dich ein und lege eine neue App an.
  2. Lege einen Namen für einen Ordner fest, in welchem die Bilder gespeichert werden sollen. Die Bilder können später auch über das Web oder eine andere App in diesen Ordner geladen werden.
  3. Erstelle einen geheimen Zugangsschlüssel für den Zugriff auf diesen Ordner (App-Tooken). Dieser Schlüssel muss im Bereich Bilder auf dieser Seite eingetragen werden.

Achtung: Gib diesen Schlüssel an niemanden weiter!

Modul News

Du benötigt einen individuellen Schlüssel. Für eine private Nutzung des Moduls kann dieser kostenfrei erstellt werden:

  1. Gehe zu der Seite https://newsapi.org/ und loge Dich ein und lege eine neue App an.
  2. Lege einen Namen für einen Ordner fest, in welchem die Bilder gespeichert werden sollen. Die Bilder können später auch über das Web oder eine andere App in diesen Ordner geladen werden.
  3. Erstelle einen geheimen Zugangsschlüssel für den Zugriff auf diesen Ordner (Key). Dieser Schlüssel muss im Bereich News auf dieser Seite eingetragen werden.

Achtung: Gib diesen Schlüssel an niemanden weiter!
